# Kinect Camera Operator

Recording tool for the Azure Kinect DK. Double-click `operator.exe` — no terminal knowledge needed.

## Quick start

1. Plug the Kinect into a **USB 3** port (blue).
2. Double-click **`operator.exe`**.
3. Press **1** to record. Enter a take number (e.g. `1`), press Enter.
4. Press **Q** to stop. The file saves automatically.

Files are named `take-01-20260622-175023.mkv` and saved to the configured output folder.

## Menu options

| Key | Action |
|-----|--------|
| `1` | Record (press Q to stop) |
| `2` | Timed record (auto-stops) |
| `3` | Change recording preset |
| `4` | Set output folder |
| `q` | Quit |
